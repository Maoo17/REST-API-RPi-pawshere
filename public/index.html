API Documentation

This is an API-created for supporting an applikation for tracking cats. It is built in node.js and express and hosted on a debian server.

API-keys

Verify API-key

GET /verify_key?key=[API_KEY]

Result

{
    "message": "Key is registered as valid API-key"
}

POST Register Cat

POST /cat?api_key=[API_KEY]

Parameters

tag_id
owner_id
name
home_group
is_chipped
owner
home

DELETE cat

DELETE /cat?api_key=[API_KEY]

Parameters

tag_id

Errors

}
    "Error": {
        "failed": "Cannot delete cat.",
        "message": "Entry does not exists."
    }
}

Users

Register user

POST /users?api_key=[API_KEY]

Parameters

username
password

Deleter user

DELETE /users?api_key=[API_KEY]

Parameters

id

Login

Check Login

POST /login?api_key=[API_KEY]

Parameters

username
password

Errors

{
    "message": "User or password is wrong."
}

Success

{
    "message": "Logged in as [username]"
}
